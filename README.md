## Quickstart (Backend)
- Crear entorno virtual.
```bash
cd /path/to/project
python3 -m venv env
```

- Activar el entorno.
```bash
source env/bin/activate
```

- Instalar dependencias.
```bash
pip install -r requirements.txt
```

- Run.
```bash
python3 main.py
```
